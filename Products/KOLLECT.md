# KOLLECT

## Product Overview
KOLLECT is Predixion AI's AI-powered collections automation platform. It automates outbound dialling on overdue accounts across DPD buckets, increasing recovery rates while reducing agent workload in BFSI collections teams.

## Core Capabilities
- Automated outbound dialling on DPD 0-30, 31-60, 61-90, and 90+ buckets
- AI-scripted conversations tailored by DPD segment and borrower profile
- Real-time recovery rate tracking per DPD bucket
- Core banking system integration (via REST API / file-based)
- Escalation to human agents for negotiation or dispute cases

## DPD Bucket Reference
| Bucket | Days Past Due | Priority |
|---|---|---|
| 0-30 | Early delinquency | High volume, soft tone |
| 31-60 | Active delinquency | Primary KOLLECT focus |
| 61-90 | High risk | Escalation scripts |
| 90+ | NPA risk | Legal/agency handoff |
