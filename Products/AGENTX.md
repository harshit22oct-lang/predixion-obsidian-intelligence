# AGENTX

## Product Overview
AGENTX is Predixion AI's next-generation autonomous agent platform — designed for complex, multi-step enterprise workflows that go beyond single-turn voice or text interactions. AGENTX agents can plan, execute, and adapt across tools, data sources, and communication channels.

## Core Capabilities
- Multi-step task execution with memory and context retention
- Tool use: CRM reads/writes, calendar scheduling, document generation
- Human-in-the-loop checkpoints for high-stakes decisions
- Role-configurable agents (collections agent, onboarding agent, compliance agent)
- Audit trail for every agent action — critical for BFSI regulatory compliance

## Differentiator vs. VOIZ/KOLLECT
| Dimension | VOIZ/KOLLECT | AGENTX |
|---|---|---|
| Interaction type | Single-turn voice | Multi-step autonomous workflow |
| Human oversight | Escalation only | Configurable checkpoints |
| Use case scope | Calls / dialling | Cross-tool enterprise tasks |
| Maturity | Production | Early access / pilots |

## Current Status
- In internal development — early access pilots planned for Q3 2026
- Not yet in active client deployment

## Target Segments
- BFSI: Autonomous loan processing, compliance workflows
- Enterprise SaaS: IT helpdesk automation, onboarding agents
